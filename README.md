# EduEase AJK
Smart Tools for Teachers (AJK)

### Features
- Question Paper Generator
- Lesson Planner
- Student Records & Performance
- Digital Diary
- Bilingual Urdu/English UI
- Print / Export to PDF
- Works Offline via LocalStorage

### Deploy to GitHub Pages
1. Push this folder to a repo named `eduease-ajk`
2. In repo settings → Pages → enable for main branch / root
3. Visit: https://<username>.github.io/eduease-ajk/
